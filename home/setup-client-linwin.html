• NOTE: Usage of the verbs directory and folder are used interchangeably in this document. Apply the correct verb usage depending on if you are installing under Linux or Windows.

Requirements (specific for Linux Installations)

  1. Admin/superuser account access on the machine
  2. Wine or Proton installed on the machine
  3. At least 10GB available (for game installation)
  4. Know how to set up Windows environment-based games/apps in Wine or Proton.

 

Requirements (specific for Windows installations)

  1. Administrator account/permissions on the machine
  2. Windows 10/11 (or newer) installed on the machine
  3. At least 10GB available (for game installation)
  4. Know how to use Windows File Explorer
  5. Know how to run .exe files as an administrator
  6. Know how to use Windows Powershell as an administrator
  7. Know how to use Windows Command Prompt as an administrator

 

Game - Installation Steps
  1. Download the Final Fantasi XI/PlayOnline installer files (download all of them to the same directory/folder!).
  2. Run FFXIFullSetup_US.part1.exe to decompress the installer files.
  3. Run FFXISETUP.exe after decompression completes to begin installing Final Fantasy XI (FFXI) and PlayOnline (POL).
  4. To avoid possible future issues with allowing FFXI to access/modify its own files, it is best to install FFXI and POL to a directory/folder of its own on your computer (we'll use C:\PlayOnline\ as an example).
  5. Make sure to install everything that prompts on the installer.
  6. Once you are done with the installation we can move to the setup phase.

 

PlayOnline Setup
  • Open PlayOnline Viewer. If you have never used PlayOnline before, it will prompt you update PlayOnline, do so and make sure to restart PlayOnline after it updates.
  • Choose For PlayOnline Members!.
  • Go through the prompts using your PlayOnline account you have registered previously. If you don't have an account, you can just enter a faux one that is 4 Letters, followed by 4 numbers (i.e., you can enter ABCD1234 for the Member Name and PlayOnline ID.
  • It is not required to provide a PlayOnline Password or a SquareEnix ID, so skip entering those.
  • Close PlayOnline..

 

Game - Update Steps (semi-automatic)
  • NOTE: If you installed under Linux, or if you encounter errors/issues with any of the following steps in this section, STOP HERE and/or skip down to the steps in the "Game - Update Steps (manual)" section below.
  • Download the update_xi.ps1 script (note the location it was downloaded to, you'll need that shortly).
  • Open Powershell as an administrator.
  • Run powershell.exe -ExecutionPolicy Bypass to allow running the script.
  • Navigate to the directory/folder that update_xi.ps1 was downloaded to.
  • Run .\update_xi.ps1. Playonline should start automatically after this script completes.
  • If Playonline didn't automatically start, open Playonline once more and choose Check Files.
  • On the next screen, make sure you change the Playonline-compatible software choice to FINAL FANTASY XI.
  • Click on Check Files to repair the files.
  • It should tell that a file is missing and that it needs repair, followed by a long download and install.
  • Once it's done repairing files, close Playonline. We can move on to the “Game - First Run” step.

 

Game - Update Steps (manual)
  • NOTE: Start here if you installed under Linux, or if you encounter errors/issues with any of the following steps in previous section for section, "Game - Update Steps (semi-automatic).
  • Navigate to the directory/folder that you installed FFXI/POL into (as mentioned above, we'll use C:\PlayOnline\ as an example).
  • Go into the SquareEnix sub-directory/sub-folder followed by the FINAL FANTASY XI sub-directory/sub-folder (for full clarity, again using the above directory as our example, the path to the directory/folder you should be at would be C:\PlayOnline\SquareEnix\FINAL FANTASY XI).
  • Create a new file called file.txt.
  • Open file.txt with some type of text editor, such as Windows Notepad or Notepad++ (if installed under Windows) or Vi/VIM editor, Nano editor, Gedit editor (if installed under Linux), or any other editor of your choice for whichever operating system you are using.
  • Populate file.txt with the info below. Make sure to also add a 3rd empty line!
123456:654321:FTABLE.DAT
::
  • Save the file and then exit out of the text editor.
  • Open Playonline once more and choose Check Files.
  • On the next screen, make sure you change the Playonline-compatible software choice to FINAL FANTASY XI.
  • Click on Check Files to repair the files.
  • It should tell that a file is missing and that it needs repair, followed by a long download and install.
  • Once it's done repairing files, close Playonline. We can move on to the “Game - First Run” step.

 

Game - First Run
  1. To run FFXI for the first time (and all subsequent times), you have a choice of three "bootloaders" that are used to load the game and connect to the XIWeb game server:

 

XILoader (the barebones bootloader, loads the game and nothing else, best for testing out issues before reporting them)
  1. Download xioader.exe into a directory of its own.
  2. Open Command Prompt as an administrator.
  3. Navigate to the directory/folder where you downloaded xiloader.exe.
  5. Run xiloader.exe as follows: xiloader.exe --server xiweb.catsangel.com
  6. If this is your first time playing on XIWeb, you will need to create an account and a password and use such to connect to XIWeb.
  7. If you already have an account on XIWeb, you need to use the account and password you created to connect to XIWeb.

 

Ashita (a bootloader that includes a variety of addons/plugins to enhance your playing experience) (more geared towards players on private servers like XIWeb)
  1. Download Ashita.exe into a directory of its own.
  2. Run Ashita.exe as an administrator. Ashita will download all other necessary files and then present you with the Ashita launcher.
  3. Download xiloader.exe into the ffxi-bootmod sub-directory/sub-folder of your Ashita directory/folder.
  4. In the Ashita launcher, click on the + icon in the bottom-left of the Ashita Launcher window.
  5. Click on the New Configuration that was just created then click on the Pencil icon (that is to the right of the + icon) in the bottom-left of the Ashita Launcher window.
  6. Give your new configuration a name (xiweb or whatever name you wish).
  7. For the File: line, use: .\\ffxi-bootmod\\xiloader.exe.
  8. For the Command: line, use: --server xiweb.catsangel.com.
  9. Click on the icon in the bottom-center of the window to save your changes (the icon looks like a sheet of paper with a checkmark).
  10. Double-click on the name that you gave for the configuration you just made (in the Ashita launcher) to launch the bootloader.
  11. If this is your first time playing on XIWeb, you will need to create an account and a password and use such to connect to XIWeb.
  12. If you already have an account on XIWeb, you need to use the account and password you created to connect to XIWeb.

 

Windower (a bootloader that includes a variety of addons/plugins to enhance your playing experience) (more geared towards players on retail FFXI but can also be used for private servers like XIWeb)
  1. Download Windower.msi into a directory of its own.
  2. Run Windower.msi as an administrator. Windower will install to a directory/folder of your choice (best to use the directory/folder you downloaded Windower.msi into).
  3. Navigate to the directory/folder you installed Windower into and run Windower.exe as an administrator. Windower will download all other necessary files and then present you with the Windower launcher.
  4. In the Windower launcher, click on the + in the bottom-left of the Ashita Launcher window to create a new profile.
  5. Give the new profile a name (click on the New Profile 1 and change it to a new name of your choice) and then click on the left-pointing arrow in the upper-left corner of the profile. (Make sure to remember the name you gave the profile).
  6. Close the Windower launcher.
  7. Download xiloader.exe into the PlayOnlineViewer sub-directory/sub-folder of the SquareEnix sub-directory/sub-folder of your PlayOnline directory/folder on your computer (created in the Game - Installation steps at the top of this page).
  8. An example of the above would be: C:\PlayOnline\SquareEnix\PlayOnlineViewer ← this is only an example, this may not be the path to yours, adjust as necessary!
  9. Navigate to the directory/folder that you installed Windower into and open settings.xml with some type of text editor, such as Windows Notepad or Notepad++.
  10. You'll need to add the xiloader.exe to every profile (including the profile you just created) you see in settings.xml, as well as the with your server information and login.
  11. Here is an example profile (you can copy and paste the below code block into the file if you wish):
    12.   <profile name="xiweb">
    <consolekey>Insert</consolekey>
    <args>--server xiweb.catsangel.com</args>
    <executable>DRIVE_LETTER:\path\to\xiloader.exe</executable>
  </profile>
  12. Note the path\to\ portion in the <executable> section cannot contain spaces and cannot be a relative path.
  13. An example of the full path is: <executable>C:\PlayOnline\SquareEnix\PlayOnlineViewer\xiloader.exe</executable> ← this is only an EXAMPLE, this may not be the path to yours, adjust as necessary!
  14. Make sure you save changes to the file and then exit out of the text editor completely.
  15. Launch Windower again.
  16. Double-click on the name that you gave for the configuration you just made (in the Windower launcher) to launch the bootloader.
  17. If this is your first time playing on XIWeb, you will need to create an account and a password and use such to connect to XIWeb.
  18. If you already have an account on XIWeb, you need to use the account and password you created to connect to XIWeb.
